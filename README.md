# BMI Calculator

## Description
This is a simple Python project that calculates the Body Mass Index (BMI) of a person. It takes the user's weight and height as input, computes the BMI, and categorizes the result into standard BMI categories such as Underweight, Normal weight, Overweight, and Obesity.

## How to Use
1. Run the Python script.
2. Enter your weight in kilograms (kg).
3. Enter your height in meters (m).
4. The program will display your BMI value and your weight category.

## Features
- Calculates BMI using the formula: (BMI = (weight * 703) / (height * height))
- Provides clear feedback about your weight status based on BMI.
